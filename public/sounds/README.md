# Sound Assets

This directory contains audio files for the game feedback system.

## Required Files

- `correct.mp3` - Played when user answers correctly
- `wrong.mp3` - Played when user answers incorrectly
- `complete.mp3` - Played when lesson is completed
- `click.mp3` - Played on button interactions

## Notes

- Keep files small (<50KB each) for fast loading
- Use MP3 format for broad browser compatibility
- Consider adding OGG fallbacks for better compression
